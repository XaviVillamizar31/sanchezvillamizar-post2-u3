# Dashboard CSS3

**Estudiante:** Xavi Villamizar  
**Programa:** Ingeniería de Sistemas  

## Descripción del proyecto
Este proyecto consiste en el desarrollo de un dashboard web responsivo utilizando **HTML5 y CSS3**. El diseño implementa **CSS Grid para la estructura principal de la página** y **Flexbox para la organización de los componentes internos**, como el sidebar, la barra superior y las tarjetas de estadísticas. El objetivo es aplicar modelos modernos de layout en CSS sin utilizar frameworks externos.

## Instrucciones de ejecución
1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en **Visual Studio Code**.
3. Hacer clic derecho en el archivo **index.html**.
4. Seleccionar **"Open with Live Server"**.
5. El navegador abrirá automáticamente el dashboard.
